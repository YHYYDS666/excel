# 用户手册

## 快速开始

1. 安装依赖
```bash
pip install -r requirements.txt
```

2. 运行程序
```bash
python src/main.py
```

## 基本操作

### Excel文件操作
- 打开Excel文件
- 刷新工作簿
- 选择工作区域

### 公式操作
- 查看公式列表
- 搜索公式
- 应用公式
- 管理公式

### 历史记录
- 撤销操作
- 重做操作
- 重置状态

## 常见问题

1. Excel连接失败
   - 检查Excel是否已经打开
   - 检查文件是否被其他程序占用

2. 公式应用失败
   - 确认选择了正确的区域
   - 检查公式参数是否正确

3. 历史记录问题
   - 确保文件未保存
   - 检查是否有可撤销的操作
