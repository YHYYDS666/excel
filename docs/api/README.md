# API文档

## 模块说明

### GUI模块

#### WindowManager
窗口基础管理器,提供窗口通用属性和行为管理。

#### ToolbarManager 
工具栏管理器,负责工具栏按钮创建和事件处理。

#### StatusManager
状态栏管理器,处理状态信息显示和更新。

### 工具类模块

#### ExcelManager
Excel操作管理器,处理Excel文件的打开、保存等操作。

#### FormulaManager
公式管理器,处理公式数据的读取、保存和查询等。

#### HistoryManager
历史记录管理器,处理操作历史的保存和恢复。
